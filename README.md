# Sin Yu Restaurant Website

A responsive multi-page website for **Sin Yu** Chinese-American restaurant, built as the final project for **IS117 – Introduction to Web Development (Fall 2025)**.  
The site showcases the restaurant’s history, menu, ingredients, and contact information using **HTML, CSS, and Bootstrap 5**.

## Project Structure

- `index.html` – Home page with hero section, image carousel, and featured dish cards.
- `menu.html` – Full menu with appetizers, soups, and entrées.
- `ingredients.html` – Card grid listing key dishes and their ingredients.
- `history.html` – Story of the restaurant and the history behind popular dishes.
- `about.html` – About the family/chefs and restaurant background.
- `contact.html` – Contact page with a Bootstrap form and location/hours.
- `css/main.css` – Custom styles layered on top of Bootstrap.
- `images/` or `Menu Item Pictures/`, `Logo Image/` – Restaurant logo and food photography.

## Key Features (Matches Course Rubric)

- **5+ Pages:** Home, Contact, and multiple interior content pages.
- **Bootstrap Layout:** Navbar, grid system, responsive containers, cards, and utilities.
- **Responsive Navigation:** Collapsing navbar with dropdown menu on smaller screens.
- **Cards:** Used on the homepage and ingredients page to highlight dishes.
- **Two-tier Navigation:** “Menu” dropdown with sub-pages (Full Menu, Ingredients, History).
- **Contact Form:** Includes name, email, message fields, and a submit button.
- **Responsive Design:** Images use `img-fluid` and appropriate `alt` attributes; layout adapts to smaller screens.
- **Footer:** Shared footer with copyright and links.
